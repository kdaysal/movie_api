*--Welcome-to-myFlix--*>

This is a web application to provide users with information about various movies, directors, genres, and more! Users may sign up, update personal information, and create a list of favorite movies.

Movie enthusiasts want to be able to access information about different movies, directors, and genres. The server-side of the myFlix application will ensure they have access to this information, that their requests can be processed, and that all necessary data can be stored.


Business Logic URL Method Type Query Parameters Request Body Data Format Response Body Data Format
Return a list of all movies to the user /movies GET None None A JSON object containing data for all movies
Return data (description, genre, director, image URL, featured-or-not) about a movie -by title- /movies/[title] GET :title None A JSON object containing data for the specified movie including description, genre, director, img URL, featured-or-not. For example: 
            
  {
    Title: 'Memento', 
    Description: 'Memento is a 2000 American neo-noir mystery thriller film', 
    Genre: {
      name: 'mystery',
      description: 'a novel, short story, play, or film whose plot involves a crime or other event that remains puzzlingly unsettled until the very end'
    } 
    Director: {
      name: 'Christopher Nolan', 
      birth: '1970',
      death: 'N/A'
      bio: 'Christopher Edward Nolan (born 30 July 1970) is a British-American film director, producer, and screenwriter.'
    },
    ImgURL: '',
    featured: 'Yes'
  }
Return data about a genre (description) -by genre name/title- (e.g., "Horror") /movies/genres/[genre] GET :genre None A JSON object containing data for the specified genre. For example: 
            
  { 
    name: 'Comedy', 
    description: 'a play, movie, etc., of light and humorous character with a happy or cheerful ending',
  }
Return data about a director (bio, birth year, death year) -by name- /movies/directors/[name] GET :name None A JSON object containing data for the specified director. For example: 
            
  { 
    name: 'Quentin Tarantino', 
    birth: '1963', 
    death: 'N/A',
    bio: 'Quentin Jerome Tarantino (born March 27, 1963)[2] is an American filmmaker, actor, film critic and author.'
  }
Allow new users to register /users POST None A JSON object containing data for the new user. For example: 
            
  { 
    "firstname": "George",
    "lastname": "Washington"
    "username": "firstprez", 
    "email": "georgeW@gmail.com", 
    "password": "abc123xyz!"
    "birthday": "01/22/1732", 
    "telephone": "770-707-7700"
  }
A JSON object containing data for the new user, including newly-added ID. For example: 
            
                { 
                    "id": "47816",
                    "firstname": "George",
                    "lastname": "Washington"
                    "username": "firstprez", 
                    "email": "georgeW@gmail.com", 
                    "password": "abc123xyz!"
                    "birthday": "01/22/1732", 
                    "telephone": "770-707-7700"
                  }
Allow users to update their existing user info (username) /users/[username] PUT :username A JSON object containing the updated username. For example: 
            
  { 
    "username": "oldestprez"
  }
A JSON object with the full updated user profile. For example: 
            
                { 
                    "id": "47816",
                    "firstname": "George",
                    "lastname": "Washington"
                    "username": "oldestprez", 
                    "email": "georgeW@gmail.com", 
                    "password": "abc123xyz!"
                    "birthday": "01/22/1732", 
                    "telephone": "770-707-7700"
                  }
Allow users to add a movie to their list of favorites -by movie name- /users/[username]/[movie] POST :username, :movie None A text message confirming that the movie was added
Allow users to remove a movie from their list of favorites -by movie name- /users/[username]/[movie] DELETE :username, :movie None A text message confirming that the movie was removed
Allow existing users to deregister from myFlix /users/[username] DELETE :username None A text message confirming that the user was deregistered (removed)
