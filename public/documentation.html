*--Welcome-to-myFlix--*>

This is a web application to provide users with information about various movies, directors, genres, and more! Users may sign up, update personal information, and create a list of favorite movies.

Movie enthusiasts want to be able to access information about different movies, directors, and genres. The server-side of the myFlix application will ensure they have access to this information, that their requests can be processed, and that all necessary data can be stored.

Allow users to add a movie to their list of favorites -by movie name- /users/[id]/[movieTitle] POST :id, :movieTitle None A text message confirming that the movie was added
Allow users to remove a movie from their list of favorites -by movie name- /users/[id]/[movieTitle] DELETE :id, :movieTitle None A text message confirming that the movie was removed
Allow existing users to deregister from myFlix /users/[id] DELETE :id None A text message confirming that the user was deregistered (removed)
